# 🏭 Franchise Factory — Konzept

## Was ist die Franchise Factory?
Ein dezentrales Protokoll das es ermöglicht, Franchise-Geschäftsmodelle als Smart Contracts
auf der A-TownChain zu registrieren, lizensieren und zu betreiben.

## Kernprinzipien
1. **Dezentral** — Keine zentrale Behörde, alle Regeln in Smart Contracts codiert
2. **Transparent** — Alle Transaktionen on-chain, öffentlich einsehbar
3. **Automatisiert** — Revenue-Sharing und Auszahlungen automatisch per Contract
4. **Proprietär** — Vollständig in ATCLang entwickelt, kein EVM-Fork

## Akteure
| Rolle | Beschreibung |
|-------|-------------|
| **Franchisor** | Markeninhaber — stellt Lizenzen aus |
| **Franchisee** | Lizenznehmer — betreibt Franchise |
| **Registry Owner** | Plattform-Admin |
| **FFT Holder** | Governance-Teilnehmer |

## Token Economy
- **ATC** — Basis-Currency, Lizenzgebühren
- **FFT** — Franchise Factory Token, Governance-Rechte
- **Franchise NFT** — Lizenz-Nachweis (ATC-9000)
