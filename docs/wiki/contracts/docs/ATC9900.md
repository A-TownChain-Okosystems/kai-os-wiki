# ATC-9900 Governance DAO

## Governance-Flow
1. `propose(description)` → Proposal-ID
2. `vote(voter, proposal_id, option)` → Stimme abgeben
3. `finalize_proposal(proposal_id)` → Nach Voting-Periode
4. `execute_proposal(caller, proposal_id)` → Timelock abwarten

## Parameter
- Min-Stake: 10.000 ATC (Voting-Power)
- Voting-Dauer: 7 Tage
- Quorum: 10%
- Threshold: 51%
- Timelock: 48h nach Voting
