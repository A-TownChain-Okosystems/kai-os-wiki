# ATC-8300 Fungible Token Standard

## Interface
```python
name()         -> str
symbol()       -> str
decimals()     -> int         # Standard: 18
total_supply() -> int
balance_of(addr)    -> int
transfer(to, amount) -> dict
approve(caller, spender, amount) -> dict
allowance(owner, spender) -> int
transfer_from(caller, from_, to, amount) -> dict
mint(caller, to, amount) -> dict   # Nur Owner
burn(caller, amount) -> dict
```

## Security-Features
- Overflow-Schutz: `safe_add`, `safe_sub`
- Reentrancy-Guard: BaseContract `_nonreentrant_enter/exit`
- approve() Front-Running: Erst auf 0 setzen
